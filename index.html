What

- JavaScript library for manipulating documents based on data;

- Mike Bostock, Vadim Ogievetsky, Jeffrey Heer.
Initially released in Feb. 2011;

- Stands for Data Driven Documents;

- Used web standards: HTML, CSS, SVG;

- Github - https://github.com/d3/d3

Why

- Helps you bring data to life;

- Tell your story more easily;

- Declarative and Data Bound;

- Manipulating Data based SVGs;

- Interactive;

          svg.selectAll().data(rides);

Enter > Update > Exit


         svg.selectAll()
            .data(rides)

            .update()

Scale functions


        let thresholdScale = d3.scaleThreshold()
         .domain([0, 50, 100])
         .range(['#ccc', 'lightblue', 'orange', '#ccc']);


         thresholdScale(-10) // returns '#ccc'
         thresholdScale(20) // returns 'lightblue'

Conclusion

D3.js is another JavaScript library, one of thousands, but it can also change the way we think what a web page can do. Although you may initially use it to build one-off data visualizations, D3 has much more power and functionality than that.
